发表于 2017-11-23 |

简介

Markdown是轻量级标记语言，可以兼容html语言

标题，1-6个（#）来表示1-6系标题

#一级

##二级

###三级
…

列表用(-)或者阿拉伯数字写出无序或有序列表
（-）号

  • one
  • two
  • three
    数字
    1 one
    2 two
    3 three

斜体 用一个下划线 斜体 或星号 斜体 框起来
粗体 用两个下划线 粗体 或星号 粗体

代码和代码块，使用一个英文的重音符，就是键盘1右边的那个符号框起来为代码，而在每行代码前面加上tab键或者四个空格，可以编写代码块

function f(){ var information = '这是一个代码块' }

使用三个星号 或者下划线 可以写分隔符

下划线

星号

链接 中/[文字]/(链接地址) 可以编写链接
